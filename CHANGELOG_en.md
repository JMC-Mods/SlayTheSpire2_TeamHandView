# Changelog

All notable changes to this project will be documented in this file.

Versioning rule: major.minor.patch, where the patch number is incremented after each change.

## [1.3.0] - 2026-7-9
### Added
- The previous locked state is now preserved after save/load (SL); this release raises the required JML version to 1.6.0.

## [1.2.0] - 2026-6-19
### Changed
- Adapted to the format migration for the official mod release.

## [1.1.0] - 2026-5-15
### Added
Fixed some resolution adaptation issues; initial release.

## [1.0.4] - 2026-4-30
### Added
Added a configurable shortcut combination for locking or unlocking the currently hovered remote player hand preview.

## [1.0.3] - 2026-4-30
### Fixed
Changed the reference to JmcModLib.Runtime.dll and corrected the actual resource location of the localization files.

## [1.0.2] - 2026-4-30
### Fixed
Corrected the resource directory for the JML configuration localization files.

## [1.0.1] - 2026-4-30
### Added
Show the current hand when hovering over another player's status bar in multiplayer.

## [1.0.0] - 2026-4-30
### Added
Initial version.
